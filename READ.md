# Kasi Rao — Portfolio Website

A personal portfolio website built using HTML, CSS, and JavaScript, showcasing my projects, skills, and development journey as an aspiring full stack developer.

---

## 🌐 Live Sections

* **Hero** — Introduction with name, role, and quick access to projects and resume
* **About** — Overview of my background, interests, and career goals
* **Skills** — Technologies and tools I use
* **Projects** — Showcase of my web development work with live demos
* **Contact** — Direct contact information and social links
